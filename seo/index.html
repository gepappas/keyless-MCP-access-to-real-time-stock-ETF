❓ Frequently Asked Questions

How do I get real-time stock prices in Claude?

Install mcprice MCP server locally with uv run --with fastmcp,httpx,yfinance,pandas python app.py or add the remote URL to your claude_desktop_config.json. Then ask Claude: "What is the current NVDA price?" and it fetches live data instantly. No API key needed.

Can I get RSI and MACD buy/sell signals inside Claude?

Yes. The technical_signals tool (new in v3.0) computes RSI-14, SMA20, SMA50, EMA9, and MACD (12/26/9) for any stock using yfinance historical data. It returns a full buy/sell signal with an overall verdict: STRONG BUY, MILD BUY, NEUTRAL, MILD SELL, or STRONG SELL. Ask Claude: "Is AAPL a buy right now? Show RSI and MACD."

What is the Fear & Greed index and how do I use it in Claude?

The Fear & Greed Index measures market sentiment from 0 (Extreme Fear) to 100 (Extreme Greed). Historically, scores below 25 represent the best long-term entry points for stocks. mcprice's fear_greed_index tool fetches the current score from alternative.me with no API key and returns an automated trading bias signal. Ask Claude: "What is the Fear & Greed index today?"

How do I track SEC insider buying in Claude or Cursor?

Use the insider_flow_scan tool. It fetches live SEC Form 4 data (updated every 2 hours via GitHub Actions), detects cluster buys when multiple insiders buy the same stock simultaneously — one of the strongest bullish signals — and flags which stocks are immediately tradeable on Revolut. Ask Claude: "Are there any cluster insider buys I can trade on Revolut today?"

How do I get Binance funding rates for crypto in Claude?

The crypto_funding_rates tool fetches live perpetual futures funding rates from Binance. A high positive rate means longs are paying heavily — a crowded trade and potential squeeze signal. Negative rates mean shorts are paying — a contrarian buy signal. Ask Claude: "What are BTC and ETH funding rates? Is there long squeeze risk?"

What stocks are available on Revolut?

Revolut offers 4,000+ NYSE and NASDAQ-listed stocks including Apple (AAPL), Tesla (TSLA), Nvidia (NVDA), Meta (META), and many more. Use the revolut_price_check or revolut_sector_scan tools in mcprice to instantly verify any stock's Revolut availability with a live price. Revolut itself is not publicly traded — expected IPO on Nasdaq or LSE in 2026–2027.

Is mcprice really free with no API key?

Yes, completely. All 16 tools use free public APIs: yfinance (Yahoo Finance with automatic auth handling), Binance Public API (no key for spot and futures data), alternative.me (Fear & Greed), and SEC EDGAR via GitHub Actions. No paid subscriptions, no rate limit headaches, no credit card required. The server is also self-hostable on Fly.io free tier.

How do I set price alerts in Claude using mcprice?

Use the price_alert_check tool. Pass a list of alerts with ticker, target price, and direction (above/below). Claude will fetch live prices and tell you which targets have been triggered. For example: "Has TSLA dropped below $200 or has BTC hit $100,000?" — the tool returns an instant triggered/pending verdict for each alert with a Revolut trade CTA if the target is hit.
